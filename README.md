# Тестовое задание ООО Онли

### Запуск проекта
1. Клонируйте проект
```bash
git clone https://github.com/olegus199/only_task.git
```
2. Перейдите в директорию проекта
```bash
cd only_task
```
3. Установите зависимости
```bash
pnpm i
```
или
```bash
npm i
```
4. Запустите проект
```bash
pnpm run dev
```
или
```bash
npm run dev
```
5. Перейдите по адресу
```
http://localhost:5173
```

### Стек технологий
- React
- TypeScript
- SCSS
- GSAP
- Swiper
